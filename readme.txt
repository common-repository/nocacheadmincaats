Плагин НЕКЭширования админки WordPress - перед глазами ТОЛЬКО ОБНОВЛЁННАЯ СТРАНИЦА !!
Adds the plugin disables caching in the admin well helps to lose that work in the admin panel and writing articles for the website (useful links in admin bar: working with WordPress 4.8)
